# MARS AR Foundation Providers
The MARS AR Foundation Provider wraps AR Foundation with MARS data providers.
Providers include:
- Camera Tracking
- Plane Finding
- Point Cloud
- Reference Points
- AR Raycasts
- Image Markers
- Face Tracking

Dependencies:
- `com.unity.xrtools.utils`
- `com.unity.xrtools.module-loader`
- `com.unity.mars`
